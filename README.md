# Exploding Pigs

**Exploding Pigs** is a client-server card game working on few-digits code rooms 

- [Installing](#Installing)
- [Loading](#Loading)
- [Authors](#Authors)

## Installing
```shell
cd client
npm install
cd ../server
npm install
```

## Loading 

### Server

```shell
cd server
node server.js
```

### Client

```shell
cd client
npx expo start
```
